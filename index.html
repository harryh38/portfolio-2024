PolluCheck

PolluCheck is a web-based application that provides real-time air pollution data for any country. Users can view pollution levels of various pollutants such as CO, NO2, and SO2, along with a visual map for easy interpretation. It offers health advice based on pollution levels and encourages healthier activity choices. The project was built using HTML, Node.js, React, CSS, JavaScript, and integrated APIs to fetch pollution data.
